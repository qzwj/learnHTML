jQuery对象和DOM对象的区别

jQuery对象就是通过包装DOM对象产生的对象, DOM对象就是HTML中DOM树中的元素节点.
jQuery对象无法使用DOM对象本身的任何方法, 只能使用自己的方法
例如: $("#foo").html() 和 document.getElemenetById("foo").innerHTML

jQuery对象和DOM对象的相互转换


    jQuery对象转DOM对象
    var $obj = $("#foo"); //jQuery对象
    var obj = $obj[0];    //DOM对象
    var obj2 = $obj.get(0); //DOM对象


    DOM对象转jQuery对象
    var obj3 = document.getElementById("foo");   //DOM对象
    var $obj4 = $(obj3);                         //jQuery对象

    注意:
    1.平时使用的jQuery对象都是$()函数制造的,$()就像一个函数的制造工厂
    2.DOM对象方法和jQuery对象的方式一定不要互相使用

jQuery和其他js库的兼容

    jQuery.noConflict(); // 将变量$的控制权交给其他库,自己使用jQuery变量
    jQuery("#foo");

    var $w = jQuery.noConflict(); // 自定义一个快捷变量
    $w("#foo");

    jQuery.noConflict();
    (function($){
        // 里面可以继续用$变量使用jQuery对象
        $(function{

        });
    })(jQuery);

    jQuery库在其他库之前导入, 可以放心使用jQuery对象, 把$权限交给别的库, 不用使用jQuery.noConflict();
